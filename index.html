Commands for references

The purpose making this website is to add list of linux, Vi and Git commands taught in lecture

Linux Commands

Command Use
pwd Gives the present working directory
ls Lists down the files or directories
mkdir dir_name Creates a new directory
clear Clears the current window
history Provides complete history of command used
cd dir_name Changes the directory
rm filename Removes the selected file
rm -rf foldername Removes the selected folder

Git Commands

Command Use
git init Creates a repository
git add filename Adds the file to git in preparation of commit
git commit -m "message" Does the commit to save changes
git checkout -b branchname Makes a new branch and checkout
git checkout branchname Checkout branch
git merge branchname Merge the branch Note: You have to be on the branch that you want to merge to

Vi Commands

Command Use
I Insert mode
Escape key Takes out of insert mode
w Command to write the changes
q Command to quit
wq Write and quit
Shift key + colon Command promot for vi
